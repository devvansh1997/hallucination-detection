\section{Dataset Overview}

All datasets generated with HARP-compatible beam search (10 beams, early-stopping
on first correct answer).  Hidden states extracted from all generated beams,
mean-pooled across tokens, saved as $(L, D)$ tensors.  Labels assigned via
contrastive BLEURT/ROUGE judge matching HARP's rubric.

\section{HOSVD Baseline Results (HARP Protocol)}

\textbf{Evaluation Protocol:} HARP-style known/unknown split.  Truthful
prompts split 75/25 for train/valid.  All hallucinated prompts added to
valid.  Factor matrices $U_L$, $U_D$ computed from train only (zero leakage).
Random Forest (200 trees, balanced class weights) on 320-dimensional
core tensors ($R_L=5$, $R_D=64$).

\section{Key Observations}

\begin{enumerate}
  \item \textbf{Cross-model consistency.} TruthfulQA AUROC is nearly identical
        across models (0.7717 vs 0.7743), suggesting the HOSVD subspace captures
        a model-agnostic hallucination signature.

  \item \textbf{TriviaQA is the strongest signal.} AUROC 0.8332 on LLaMA-3.1-8B
        benefits from the largest training set (15,491 beams) and most balanced
        label distribution among the four datasets.

  \item \textbf{NQ-Open at 0.8252 despite 96.2\% hallucination rate.}
        The HOSVD extracts a usable decision boundary even under extreme class
        imbalance -- only 1,158 truthful beams out of 30,140 total.

  \item \textbf{TyDiQA-GP at 0.8128 with best label balance (62.6\% hall).}
        The multilingual context passage helps LLaMA generate more accurate
        answers, improving the quality of truthful-beam features.
\end{enumerate}
